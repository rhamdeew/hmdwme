  • Отключил анимацию переключения рабочих столов в MacOS. Вдруг понял что это перемещение вправо-влево меня немного напрягает. Дошли руки это сделать. System Preferences -> Accessibility -> Display -> [X] Reduce Motion

Также временно снова перекатился в Vim/Neovim в Alacritty. VS Code конечно прекрасен, но захотелось вот снова кодить в виме)
