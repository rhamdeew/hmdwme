июль 4 2018

С недавних пор начал систематизировать знания и избавляться от всего лишнего. Часто натыкаюсь в сети на интересные сервисы, крутой опенсорсный софт на гитхабе и просто полезные статьи и быстро забываю про них. Потом долго не могу вспомнить нужную ссылку спустя некоторое время.

Считайте эту серию постов с тегом href моей попыткой запомнить все эти утилиты и сервисы.

Сервисы

unsplash.com - сайт с красивыми картинками огромного разрешения, любой тематики и с разрешением на бесплатное использование.

repl.it - онлайн-REPL для кучи языков программирования.

mailtrap.io - сервис для отладки писем с сайта. В бесплатной версии отлаживать можно только через их smtp.

mail-tester.com - сервис диагностики правильности настройки почтового сервера. Предлагает отправить письмо на сформированный уникальный адрес и выдаёт результат диагностики с подробным описанием некорректных настроек.

compressor.io - очень хорошо жмёт картинки.

Утилиты

glue - утилита для генерации спрайтов из набора картинок

ripgrep - супер быстрый аналог grep написанный на rust

fd - супер быстрый аналог find написанный на rust

tokei - супер быстрый анализатор кода написанный на rust. Показывает статистику по используемым языкам программирования в проекте (кол-во файлов, кол-во строк, процентное соотношение)

fzf - очень быстрый нечёткий поиск. Прикрутил его к терминалу чтобы по CMD+R (Ctrl+R) приятнее искалось по истории команд. Написана на Go.

exa - функциональная и быстрая замена ls написанная на rust.

Ну и все наверное. На первый раз хватит) Следующий выпуск через неделю.
