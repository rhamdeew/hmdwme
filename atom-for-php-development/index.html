Atom для PHP-разработчика

окт. 20 2015

Решил тут попробовать Atom, а его оказывается уже допилили =)
Если говорить кратко, то по моему мнению им уже можно заменить старый добрый Sublime Text. Благо что основные хоткеи тут начисто слизаны с ST.
Из коробки есть почти все необходимое для работы.

Дополнительно установил плагины:

  • api-docs - этот замечательный плагин позволяет открывать документацию (DevDocs), прямо в окне редактора всего лишь по нажатию Ctrl+D. Поддерживается все что есть в DevDocs, но нужные языки/библиотеки нужно включить дополнительно в настройках плагина.
  • atom-beautify - Ctrl+Alt+B сделает ваш код красиво отформатированным.
  • emmet - его и так все знают
  • linter - базовый плагин для разных линтеров, установился как зависимость к linter-php
  • linter-php - проверяет синтаксис php на лету
  • tabs-to-spaces - позволяет сменить в файле пробелы на табуляции и наоборот

Я очень рад =)

UPD: Большие файлы в нем все же лучше не открывать. Начинает жутко тормозить. Тяжелые XML не хочет.
